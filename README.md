## 🧠 What I do best

### 1. **Hybrid Trading Systems**
I design systems where **MT4/5 handles execution** and **Python handles intelligence** – signal generation, feature engineering, ML models, and position sizing logic.

### 2. **Smart Money Concepts (SMC) Automation**
I translate concepts like **order blocks**, **liquidity sweeps**, **CHoCH**, **BOS**, and **fair value gaps** into testable, executable MQL code with on‑chart visualization.

### 3. **CSV‑Driven & Bridge Architectures**
Projects like [SignalFileTrader PRO](https://github.com/smcalgosystems/SignalFileTrader-PRO-for-MetaTrader) demonstrate file‑based command/control between Python PM engines and MT4 – including `EXIT`, `ADD`, and `REDUCE` commands.

### 4. **Machine Learning for Trading**
I work with real market data, build features, train models (CatBoost, XGBoost, LightGBM), and integrate predictions into live trading decisions.

### 5. **Professional Documentation & UI**
Every system includes clear `README.md`, architecture docs, CSV schema definitions, and on‑chart control panels for real‑time operator supervision.
